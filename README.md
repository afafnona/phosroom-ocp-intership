#  PhosRoom – OCP Training Room Reservation System

##  Project Overview
PhosRoom is a web application developed as part of my internship at OCP Group (Youssoufia).  
The system allows users to manage training room bookings, view availability, and make reservations easily with a user-friendly interface.

##  Objectives
- Digitalize the reservation process for training rooms
- Avoid booking conflicts
- Provide a clear booking interface
- Work with a database to store user and reservation data

##  Technologies Used
- PHP – Database management
- HTML & CSS – Frontend layout and design
- JavaScript – Dynamic interaction on pages
- MySQL / SQL – Database management

##  Main Features
- User login and authentication
- View available rooms and schedules
- Create room reservation entries
- Store reservations in database
- Prevent double bookings

##  How to Run Locally

1. Install *XAMPP or WAMP* on your computer.  
2. Place the project folder inside the htdocs directory.  
3. Open *phpMyAdmin* from your browser.  
4. Create a database and import the provided database.sql file.  
5. Start Apache & MySQL from the control panel.  
6. Open your browser and go to:
http://localhost/phosroom-ocp-intership/
